# Destini

Интерактивная игра с выбором сюжета. Реализация MVC паттерна.

## Функционал

- 6 сюжетных веток
- Выбор влияет на развитие истории
- 3 возможные концовки
- Адаптивный интерфейс
- Кнопки с закруглёнными углами

## Технологии

- Swift
- UIKit
- MVC (Model-View-Controller)
- Storyboard
- Struct (Story, StoryBrain)

## Структура MVC

- **Model**: Story.swift, StoryBrain.swift
- **View**: Main.storyboard
- **Controller**: ViewController.swift

## Сюжет

История начинается с поломки машины на пустынной дороге. Вам предстоит принимать решения, которые приведут к разным концовкам.

# Destini

Interactive story game with choices. MVC pattern implementation.

## Features

- 6 story branches
- Choices affect story development
- 3 possible endings
- Responsive interface
- Buttons with rounded corners

## Technologies

- Swift
- UIKit
- MVC (Model-View-Controller)
- Storyboard
- Struct (Story, StoryBrain)

## MVC Structure

- **Model**: Story.swift, StoryBrain.swift
- **View**: Main.storyboard
- **Controller**: ViewController.swift

## Story

The story begins with a car breakdown on a deserted road. You must make decisions that lead to different endings.
